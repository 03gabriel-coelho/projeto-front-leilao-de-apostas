## O que foi desenvolvido?

Foi desenvolvido uma aplicação simulando como poderia ser o front-end de um leilão de apostas online.

## Tecnologias utilizadas:

- HTML
- JavaScript
- CSS
- React.js
- Bootstrap
- ESLINT
- FireBase
